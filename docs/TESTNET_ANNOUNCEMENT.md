**Q: Is testnet RNR worth anything?**
A: No, TEST-RNR has no monetary value. It's for testing only.

**Q: Will my testnet balance transfer to mainnet?**
A: No, mainnet will start fresh. Testnet is separate.

**Q: Can I mine on multiple machines?**
A: Yes! Run multiple nodes to increase your chances.

**Q: What happens if I find a bug?**
